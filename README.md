# Старт приложения

## Инициализация pre-commit

```
 pre-commit install 
```

## Docker

### Создание секретного файла

Необходимо создать .env с содержимым по аналогии с env.example

### Сборка и старт контейнера

```
make build
make up
```

## Локальный запуск

### Установка зависимостей  (необходим python 3.12+)

```
poetry install
```

### Локальный старт

Запуск файла main.py
